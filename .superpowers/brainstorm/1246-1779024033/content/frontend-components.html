关键设计决策

工具包隔离

每个工具包有自己的 components/ 和 api/ 目录，互不依赖。导航栏只读取每个工具包的注册信息（名称、图标、路由）

子待办也计入完成统计

周视图/月视图遍历所有待办的子待办，完成的子待办和完成的父待办一起列出，确保 AI 总结能看到完整的进度

状态管理：React Context + useReducer

对于这个规模的项目，不需要 Redux。Context 足够管理当前工具包状态，数据通过 API 与后端同步
