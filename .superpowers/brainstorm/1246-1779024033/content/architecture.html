如何添加新工具包

每个工具包是一个独立文件夹，放在 tools/ 目录下，自动注册到导航栏：

tools/
├── todo-summary/          ← 待办 & 总结（第一个工具包）
│   ├── index.ts           ← 导出：名称、图标、路由、组件
│   ├── components/        ← 该工具包的 React 组件
│   └── api/               ← 该工具包的后端路由
├── future-tool-2/         ← 未来的工具包，照这个模板来
└── future-tool-3/

添加工具包只需：新建文件夹 → 实现接口 → 放到 tools/ 目录 → 自动出现在导航栏
